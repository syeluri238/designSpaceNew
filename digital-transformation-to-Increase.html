Case Study

Digital Transformation to Increase

Industry - Pharmaceuticals

RESEARCH
  • Stakeholders interviews
  • Heuristic Evaluation
STRATEGY
  • Task Flows
  • Data Visualization
DESIGN
  • User Experience
  • User Interface
  • Illustrations & Icons

Requirement

To modernise the existing legacy web applications to boost operational efficiencies and overall business performance.

Objectives

  • Ease of navigation
  • Create simple visuals and concise copy for communication.
  • UI for graphical data, case exploration, and analysis.
  • Increase compatibility and scalability
  • Improved application performance, and deliver intuitive UI with sleek features.

Problem

A top technology innovator in medical product seeks to upgrade its outdated and limited investigate system to a modern platform with real-time data analysis, automated data collection, and enhanced functionality for efficient operations. Evoke was approached for the upgrade.

Solution

Evoke's custom design process improved the legacy application's user experience through stakeholder interviews, user personas, and a modern design language. The research-based design resulted in easier tasks, improved navigation, and a more visually appealing interface.

Discover & Define

The clutter in the app caused a cognitive load on the users. The research-led insights about user interaction with the app aided us to redefine the information architecture and user interface of the app. Analysed user journey and conducted contextual/competitive research to derive a clear understanding of how the ideal user interaction must be and ways to achieve it. Actively worked to receive every key stakeholder aligned with the redevelopment process and how it will impact the workflows.

Ideate, Design & Deliver.

Various approaches of viewing and traversing the data grids, applying filters, and locating the precise data were devised and presented based on the thoughts gleaned from the brainstorming session.
